### 開発環境

* Vite
* PostCSS
* Tailwind
* Pug
* local by Flywheel

### ディレクトリ構成

* src/ -> ソースファイル
* local/ -> WordPress
* wp_export/ -> All-in-One WP Migration の書き出しデータ

### コマンド

* npm run dev -> viteで静的サイト実行
* npm run build -> viteで書き出し
* npm run wp -> WP用に書き出し（watch）
* npm run wp:build -> WP用書き出し（build） ※未設定
